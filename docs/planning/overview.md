**はじめてこのリポジトリを触る方は、このファイルからお読みください。**

---

## このプロジェクトは何をするものか

スマホゲームの「施設取り合いバトル」の作戦を立案するためのシミュレーターです。

```
あなた（プレイヤー）が:
  ↓ どの施設に何人の部隊を置くか？を設定する
  ↓ 敵がどう配置してきたとしたら？
  ↓ 戦闘がアニメーションで進んでいく
  ↓ 最終的にどの施設を制圧できたか／できなかったかがわかる
```

これをゲームの外でブラウザ上でシミュレートすることで、ギルドの作戦会議に使えます。

## 現在のフェーズ

```
Phase 0: プロトタイプ ✅ 完了
  └─ HTMLシングルファイルで基本機能を実装済み

Phase 1: 設計確定 🔶 進行中
  ├─ 実際のゲームの計算式を確認する
  ├─ 部隊移動メカニズムを決める
  └─ 技術スタックを決める（vanilla JS継続 or React移行）

Phase 2: v1開発 ⬜ 未着手
  ├─ ゲーム仕様に合わせた戦闘モデルに更新
  ├─ 敵配置をUIから変更できるようにする
  ├─ モバイル対応
  └─ 施設数・配置のカスタマイズ

Phase 3: チーム共有 ⬜ 未着手
  ├─ GitHub Pages等でホスティング
  └─ 作戦パターンの保存・共有機能
```
